# PD_amina_alysheva
## Projekta nosaukums
Kalkulators reizinājuma aprēķiniem no ievadskaitļiem
## Apraksts
Šī programma piedāva lietotājam ievadīt divus skaitļus, un pēc tam programma apreķina to reizinājumu.
## Palaišanas instrukcija
1) Lejupieladēt failu programma.py
2) Ielaid failu Python vidē
3) Palaist konsoli ar F5
4) Programma piedāva ievadīt nepieciešamus datus
